# Nigeria Economic Data Analysis

## Project Overview
This project analyzes key economic indicators of Nigeria to uncover trends and insights that can help understand economic performance.

## Objectives
- Explore Nigeria's economic indicators
- Identify economic trends
- Visualize patterns in GDP and inflation
- Generate insights useful for business and policy decisions

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
World Bank Nigeria Economic Indicators Dataset
